Tri à bulles

Écrire une fonction triBulles qui trie un tableau tab de taille n par ordre croissant.

Utiliser une boucle externe d'indice i et, à l'intérieur, une boucle interne d'indice j qui parcourt les éléments non encore triés.

Comparer chaque paire voisine : si tab[j] est strictement supérieur à tab[j+1], échanger les deux éléments. Répéter, puis retourner tab.
